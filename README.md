# 🧮 DemoApp - TDD Calculator using .NET

## 📌 Overview
This project demonstrates the basics of **Test Driven Development (TDD)** using .NET Core and MSTest.

It includes a simple Calculator application with unit tests to verify functionality.

---

## 🚀 Features
- Multiply two numbers
- Add two numbers
- Subtract two numbers
- Divide two numbers
- Handles divide-by-zero exception
- Unit testing using MSTest

---

## 🛠️ Technologies Used
- .NET Core
- C#
- MSTest Framework
- Visual Studio

---

## 📂 Project Structure
DemoAppSolution/
│
├── DemoApp/             # Main application
│   └── Program.cs
│
├── DemoApp.Tests/       # Unit tests
│   └── Test1.cs

---

## ▶️ How to Run the Project
1. Open solution in Visual Studio
2. Press `Ctrl + F5` to run the console app
3. Output will be displayed in console

---

## 🧪 How to Run Tests
1. Go to **Test → Test Explorer**
2. Click **Run All**
3. All tests should pass successfully

---

## 🧠 TDD Approach Used
- First wrote test cases
- Then implemented methods
- Verified output using assertions

---

## ⚠️ Exception Handling
- Division by zero throws `DivideByZeroException`
- Properly tested using unit tests
